# SimpleMongo

MongoDb-like document database, in memory implementation using JavaScript/NodeJs.

## Installation

Via npm on Node:

```
npm install simplemongo
```


## Usage

Reference in your program:

```js
var sf = require('simplemongo');
```

## Development

```
git clone git://github.com/ajlopez/SimpleMongo.git
cd SimpleMongo
npm install
npm test
```
